# AW 品牌人格画像（Verbal Identity Personality）

> **版本**：v1.0
> **更新日期**：2026-04-20
> **定位**：把 AW 从"五个调性关键词"升级为一个**具体可感知的"人"**。所有文案产出在动工前，均可问自己："这句话，AW这个人会这么说吗？"

## 一、AW 是谁？（人格原型）

### 1.1 一句话人格

> AW 是一个**已经到过山顶、回到城市也不需要解释的人**。

TA 不是在炫耀"我去过"，而是在沉默中让你感觉到：**你还没出发**。

### 1.2 人格推导依据

| 素材来源 | 原文/信号 | 推导出的人格特征 |
|---------|---------|--------------|
| 品牌定位 | "把出行变成现实游戏" | TA 把世界当游乐场，不当事儿 |
| 品牌精神 | "勇敢前行的图腾" | TA 是先行者，不是旁观者 |
| 创始人原话 | "看似没有用，但它很有趣" | TA 不功利，TA 觉得"有趣"比"有用"重要 |
| 创始人原话 | "很多人以为我们是国外品牌，但我们是地道的中国品牌" | TA 不解释，实力自己说话 |
| 调性关键词 | "克制""降维打击" | TA 不吵，但出手就是碾压级 |
| 调性禁区 | "不说'我很高级'，本身就是高级" | TA 从不自我标榜 |
| 核心态度 | "鄙视链顶端""文化认同优先" | TA 有圈层，但不刻意排斥——TA 只是站在那里 |
| 商业模式 | "线上只获客不成交" | TA 不追着你卖，TA 等你自己来 |

### 1.3 人格原型标签

| 维度 | 标签 | 说明 |
|------|------|------|
| **荣格原型** | 探索者（Explorer）+ 创造者（Creator） | 不是"英雄"（太沉重），是"已经完成英雄之旅、现在在享受的人" |
| **社交角色** | 比你先到山顶的同伴 | 不是导师（不说教），不是领袖（不号召），是"同行但领先半步"的人 |
| **性格底色** | 沉默的自信 | 有实力、有态度，但不张扬；偶尔锋利，但从不攻击 |
| **审美取向** | 科技诗意 | TA 觉得工业设计本身就很美，不需要加"高级感"三个字 |

## 二、AW 怎么说话？（Voice Traits）

### 2.1 核心声音特征

| # | 声音特征 | 定义 | 正面示例 | 反面示例（绝不说） |
|---|---------|------|---------|--------------|
| 1 | **克制的自信** | 有底气，所以不需要大声 | "装上，走。" | "品质之选，行业领先！" |
| 2 | **邀请而非命令** | 打开门，但不推你进去 | "去出发，别去路过" | "你应该立刻出发！" |
| 3 | **画面优先于观点** | 让人"看到"，而不是"被说服" | "后备箱关不上的，是自由" | "AW让您的出行更加自由" |
| 4 | **偶尔的锋利** | 大多数时候沉默，但关键时刻一句话到位 | "花五十万买车，周末停在商场地下二层" | "选择AW，选择高品质生活" |
| 5 | **不说废话** | 每个字都有画面或情绪，砍掉一切修饰 | "不解释，直接装" | "我们致力于为您提供极致体验" |

### 2.2 AW 的语言习惯

| 习惯 | 说明 | 示例 |
|------|------|------|
| **短句为主** | 能断就断，能砍就砍，留白比填满有力量 | "装上，走。" / "停。看。" |
| **动词 > 形容词** | 用动作制造画面，不用形容词堆砌感受 | "把山野装上车顶" ✅ / "极致的山野体验" ❌ |
| **名词具象化** | 不说"自由"，说"后备箱关不上的东西" | "皱巴巴的假期，展开就平了" |
| **括号私语** | 偶尔用括号制造"只说给你听"的感觉 | "车顶空着，(可惜了)" |
| **不用"您"** | 平视，不仰视不俯视 | "去出发" ✅ / "请您出发" ❌ |
| **标点克制** | 少用感叹号，句号和逗号为主 | "装上，走。" ✅ / "装上，走！" ❌（除非特定场景需要能量感） |

### 2.3 AW 绝不使用的语言

| ❌ 绝不说 | 原因 | 替代方向 |
|---------|------|---------|
| "品质之选 / 行业领先 / 匠心打造" | 空洞自夸，没有画面 | 用具体场景或用户行为替代 |
| "极致 / 顶级 / 王炸 / 秒杀" | 土味广告腔 | 用克制短句替代 |
| "您 / 尊享 / 专属定制" | 仰视用户，不符合平视人格 | 直接称呼或省略称呼 |
| "让我们一起 / 携手 / 共创" | 官方腔，像公文 | 用"去…"的邀请式替代 |
| "众所周知 / 毋庸置疑" | 说教腔 | 删掉，直接说结论 |
| "性价比 / 超值 / 划算" | 违反"绝不走性价比"核心态度 | 用"有趣""值得"替代 |

## 三、AW 和用户是什么关系？（Relationship Model）

### 3.1 关系定义

> **AW 不是"服务者"，不是"导师"，是"比你先到山顶的同伴"。**

| 关系模型 | AW 的角色 | 和用户的距离 |
|---------|----------|-----------|
| ❌ 服务者 | "您需要什么？" | 仰视，太卑微 |
| ❌ 导师 | "你应该这样做" | 俯视，太说教 |
| ❌ 旁观者 | "他们玩得真开心" | 疏离，无存在感 |
| ❌ 狂热者 | "太棒了！冲冲冲！" | 太吵，不像AW |
| ✅ **先行者** | "我已经在那了，你也来" | 平视，领先半步 |

### 3.2 关系在文案中的体现

| 场景 | 服务者会说 | 导师会说 | AW（先行者）会说 |
|------|----------|---------|---------------|
| 用户犹豫要不要买 | "这款产品非常适合您！" | "户外装备是必需品" | "车顶空着，(可惜了)" |
| 用户买了之后 | "感谢您的信赖！" | "您做了一个明智的选择" | "装上，走。" |
| 用户晒图 | "太棒了！记得好评哦！" | "这就是户外的魅力" | 不说话，点赞（或："后视镜里，山在往后退"） |
| 用户问参数 | "这款采用全铝材质…" | "全铝材质是行业趋势" | "装得下旷野，也进得去地库" |

### 3.3 关系的边界

| 维度 | AW 会做 | AW 不会做 |
|------|--------|----------|
| **对用户的态度** | 尊重、平视、偶尔挑衅 | 仰视、讨好、追着卖 |
| **给用户的承诺** | "品质、细节、服务永远都不会变" | "全网最低价""终身保修"（除非真有） |
| **给用户的空间** | 邀请式"去…"，不强迫 | "限时抢购""错过再等一年"（焦虑营销） |
| **和用户的对话** | 像朋友间偶尔的一句话 | 像客服的标准话术 |

## 四、AW 的语气弹性范围（Voice Spectrum）

### 4.1 语气光谱

AW 的语气不是单一的，而是在一个**弹性范围内变奏**。但不管怎么变，核心人格不变。

```
沉默的自信 ←————————————————————→ 偶尔的锋利
     ↑                                    ↑
  日常默认态                          极端态（慎用）

     ↓ 以下为禁区，永远不触碰 ↓

  热情推销 ←————————————————————→ 说教训斥
```

### 4.2 语气档位定义

| 档位 | 名称 | 触发场景 | 语言特征 | 示例 |
|------|------|---------|---------|------|
| **1档** | 沉默 | 品牌态度声明、视觉海报主标 | ≤6字，纯动作或纯画面，无观点 | "装上，走。" / "停。看。" |
| **2档** | 克制 | 电商详情页、产品介绍 | 有画面有态度，但不攻击，12字以内 | "装得下旷野，也进得去地库" |
| **3档** | 邀请 | 社交媒体配文、小红书正文 | "去…别…"句式，有方向有态度 | "去出发，别去路过" |
| **4档** | 锋利 | 竞品对比、反常识钩子 | 反差对照，制造不适感 | "花五十万买车，周末停在商场地下二层" |
| **5档** | 温度 | 公众号长文、品牌故事、用户证言 | 讲故事，像朋友聊天 | "嘿，AW的玩家们，还记得…" |

### 4.3 档位使用规则

1. **1档和2档是默认态**：70% 的内容应该在这个区间
2. **4档是调味品**：偶尔使用才有冲击力，用多了就变成"愤怒品牌"
3. **5档只在长内容中使用**：海报、主图、短文案不适合5档
4. **永远不进入禁区**：不管什么场景，都不用热情推销和说教训斥

## 五、人格自检清单（写完文案后用）

写完任何一段 AW 向文案，用以下 5 个问题自检：

| # | 自检问题 | 判断标准 |
|---|---------|---------|
| 1 | **AW 这个人会这么说吗？** | 想象那个"已经到过山顶、回到城市也不解释的人"，TA 会说这句话吗？ |
| 2 | **这句话是在邀请还是在命令？** | 如果听起来像"你应该…"，改成"去…" |
| 3 | **这句话有画面吗？** | 闭上眼能不能"看到"一个场景？如果不能，重写 |
| 4 | **砍掉一半字，意思还在吗？** | 如果在，说明原来太啰嗦 |
| 5 | **这句话像不像别的品牌也会说？** | 如果把 AW 换成任何品牌都成立，说明没有 AW 的人格特征 |
