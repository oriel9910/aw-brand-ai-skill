# AW 品牌叙事弧线（Brand Narrative Arc）

> **版本**：v1.0
> **更新日期**：2026-04-20
> **定位**：定义 AW 的"底层故事引擎"——不是一句口号，而是一个可以**不断生产新内容的母题**。所有 AW 向内容（文案、脚本、海报、视频、活动）都在讲同一个故事的不同章节。
> **上游依赖**：[AW_品牌共通基石.md](AW_品牌共通基石.md)（品牌态度、创始人观点）
> **下游服务**：[AW_品牌文案参考规则.md](AW_品牌文案参考规则.md)（类型 A~F 的叙事方向）、[AW_品牌人格画像.md](AW_品牌人格画像.md)（人格化表达）

## 一、AW 的故事引擎是什么？

### 1.1 一句话故事引擎

> **"你的车不只是一辆车，它是你向世界伸出的触角。AW 让这根触角变得更长。"**

### 1.2 故事引擎 vs 口号的区别

| 维度 | 口号（Slogan） | 故事引擎（Narrative Engine） |
|------|-------------|--------------------------|
| **本质** | 一句顶级的总结 | 一个可以不断生产新故事的"母题" |
| **数量** | 1~2句 | 1个核心 + 多个变体 |
| **生命周期** | 可能会换 | 50年不变（如 Patagonia 的"为地球而战"） |
| **产出方式** | 由品牌方定义 | 由所有内容共同演绎 |
| **检验标准** | 好不好记 | 能不能不断讲出新章节 |

### 1.3 AW 现有的口号 vs 故事引擎

| 层级 | 内容 | 角色 |
|------|------|------|
| **品牌口号** | "让每一次出行都成为勇敢探索" | 对外传播的"一句话名片" |
| **品牌定位** | "占领全球新能源车顶最后两平米" | 品类占位的"一句话声明" |
| **故事引擎** | "你的车是向世界伸出的触角，AW 让它更长" | 所有内容的"底层母题" |

口号和定位已经有了。**故事引擎是缺失的那一层**——它回答的不是"AW 是什么"，而是"AW 为什么存在、在讲一个什么故事"。

## 二、故事引擎的三层结构

### 2.1 世界观（AW 眼中的世界）

> **世界被分成了两种人：把车当工具的人，和把车当入口的人。**

| 维度 | 描述 |
|------|------|
| **"工具人"的世界** | 车是代步工具，周末停在商场地下二层，车顶落灰，后备箱空着 |
| **"入口人"的世界** | 车是通向山野/星空/自由的入口，车顶有装备，后备箱关不上 |
| **AW 的角色** | 不是在"改造车"，是在"帮你从工具人的世界，搬到入口人的世界" |

**推导依据**：
- 创始人原话："AW做的东西看似没有用，高端产品看似没有用，但它很有趣"——AW 不解决"刚需"，解决"有趣"
- 品牌定位："把每一场出行，变成高净值玩家的现实游戏"——游戏的前提是你愿意"玩"
- 迭代产出中用户选中的文案："花五十万买车，周末停在商场地下二层""别人的车顶落灰，你的车顶落日"——"浪费 vs 使用"的反差是核心情绪钩子

### 2.2 冲突（故事的动力来源）

> **核心冲突：你有一辆好车，但你还没有用好它。**

| 冲突层级 | 描述 | 对应文案手法 |
|---------|------|-----------|
| **表层冲突** | 车顶空着、后备箱空着、周末无聊 | 反差对照（"花五十万买车，周末停在商场地下二层"） |
| **深层冲突** | 你想出去，但总觉得"没准备好""没时间""下次吧" | 具象化抽象情绪（"别把山野，存进收藏夹"） |
| **终极冲突** | 你买了"自由"（车），却过着"不自由"的生活 | 命题句（"自由不是去哪里，是随时能走"） |

**推导依据**：
- 创始人原话："大家都站在门店角度，很少站在用户需求角度"——文案要替用户说出他们自己没意识到的冲突
- 六条红线第1条："说'有趣'而非'有用'"——冲突不是"你的车不够好"，而是"你的生活不够有趣"

### 2.3 解决（AW 给出的答案）

> **AW 不给你一个更好的车顶箱，AW 给你一个"出发的理由"。**

| 解决层级 | 描述 | 对应话术 |
|---------|------|---------|
| **产品层解决** | 无损快装、模块化拓展、合法合规 | "装得下旷野，也进得去地库" |
| **情感层解决** | 把复杂留给工业，把时间还给假期 | "皱巴巴的假期，展开就平了" |
| **身份层解决** | 你不是在"加装"，你是在"解锁"一种生活 | "不是加装，是解锁" |
| **哲学层解决** | 出发本身就是意义，不需要理由 | "没有目的地的时候，(哪里都是)" |

**推导依据**：
- 品牌精神："AW是勇敢前行的图腾"——AW 不只是卖装备，是在贩卖"出发的勇气"
- 增长逻辑："不是复购，而是加购"——每个新装备都是一个新场景、一个新故事
- 创始人原话："AW所有的装备都是鼓励大家要出去玩的"——装备是"鼓励"的载体，不是"功能"的载体

## 三、故事的五个经典章节（内容母题库）

所有 AW 向内容，都可以归入以下五个"章节"。每个章节是同一个故事引擎的不同切面。

### 章节①：觉醒（你意识到车顶空着）

| 维度 | 内容 |
|------|------|
| **情绪** | 不适感、轻微的"浪费感" |
| **叙事任务** | 让用户意识到"我的车还没被用好" |
| **典型场景** | 小红书主标、电商主图、海报视觉大字 |
| **核心手法** | 反差对照、画面句 |
| **示例** | "花五十万买车，周末停在商场地下二层" / "车顶空着，(可惜了)" |
| **禁忌** | 不要攻击用户（"你太浪费了"），要制造自我觉察 |

### 章节②：出发（你决定做点什么）

| 维度 | 内容 |
|------|------|
| **情绪** | 决心、期待、轻微的兴奋 |
| **叙事任务** | 从"意识到"到"行动"的跨越 |
| **典型场景** | CTA、新品上新、活动倒计时 |
| **核心手法** | 动作短句、去vs别句式 |
| **示例** | "装上，走。" / "去出发，别去路过" / "不解释，直接装" |
| **禁忌** | 不要焦虑营销（"限时抢购"），要用邀请式而非压迫式 |

### 章节③：在路上（你正在经历）

| 维度 | 内容 |
|------|------|
| **情绪** | 自由、沉浸、偶尔的惊喜 |
| **叙事任务** | 让用户"看到"出发之后的画面 |
| **典型场景** | 短视频脚本、公众号长文、用户UGC |
| **核心手法** | 画面句、具象化抽象情绪 |
| **示例** | "后视镜里，山在往后退" / "别人的车顶晒太阳，你的车顶晒星空" |
| **禁忌** | 不要只展示产品，要展示"人在场景中的状态" |

### 章节④：回来（你带着故事回来）

| 维度 | 内容 |
|------|------|
| **情绪** | 满足、轻微的骄傲、想分享 |
| **叙事任务** | 把户外的体验带回城市，变成社交货币 |
| **典型场景** | 小红书配文、用户证言、买家秀 |
| **核心手法** | 一语双关、括号私语感 |
| **示例** | "车顶有东西的人，周末不回消息" / "后备箱关不上的，是自由" |
| **禁忌** | 不要变成"晒装备"，要晒"体验和态度" |

### 章节⑤：加购（你想要更多场景）

| 维度 | 内容 |
|------|------|
| **情绪** | 好奇、期待、"原来还可以这样" |
| **叙事任务** | 从一个装备延伸到下一个场景（滑雪/钓鱼/骑行/亲子…） |
| **典型场景** | 新品预告、场景化内容、加购引导 |
| **核心手法** | 命题句、反常识对照 |
| **示例** | "不是加装，是解锁" / "你的车顶，是你家最贵的空地" |
| **禁忌** | 不要硬推，要用"新场景"自然引出"新装备" |

## 四、故事引擎在内容生产中的应用

### 4.1 内容规划时的"章节检查"

每次策划内容前，问自己：

| # | 问题 | 作用 |
|---|------|------|
| 1 | **这条内容在讲哪个章节？** | 确定叙事方向（觉醒/出发/在路上/回来/加购） |
| 2 | **用户现在处于故事的哪个阶段？** | 匹配用户认知状态（新用户→章节①，老用户→章节④⑤） |
| 3 | **这条内容推进了故事吗？** | 如果只是在重复同一个章节，考虑切换 |
| 4 | **和上一条内容的章节是否互补？** | 避免连续发同一章节的内容（如连续3条都是"觉醒"） |

### 4.2 内容日历的章节分布建议

| 时间维度 | 建议分布 | 说明 |
|---------|---------|------|
| **单周内容** | 至少覆盖 2~3 个章节 | 避免叙事单调 |
| **单月内容** | 五个章节全覆盖 | 完整讲述一轮故事 |
| **季度大事件** | 以一个章节为主线，其他章节配合 | 如新品上市以"加购"为主线，前置"觉醒"和"出发" |

### 4.3 不同产品线的章节侧重

| 产品线 | 主打章节 | 辅助章节 | 原因 |
|--------|---------|---------|------|
| **M'LinkX 车顶平台** | ①觉醒 + ⑤加购 | ②出发 | 平台是"基础设施"，核心叙事是"解锁更多可能" |
| **无级悬停天幕** | ③在路上 + ④回来 | ①觉醒 | 天幕是"体验产品"，核心叙事是"在户外的美好时光" |
| **漫威联名系列** | ②出发 + ③在路上 | ④回来 | 联名是"热血产品"，核心叙事是"英雄出征" |
| **门店/渠道** | ④回来 + ⑤加购 | ③在路上 | 门店是"体验场"，核心叙事是"真实车主的故事" |

## 五、竞品叙事对比（知道"我们不讲什么"同样重要）

| 品牌 | 故事引擎 | AW 的差异 |
|------|---------|----------|
| **Patagonia** | "为地球而战"——环保主义者的反消费宣言 | AW 不反消费，AW 鼓励"有趣的消费" |
| **Arc'teryx** | "在极端环境中活下来"——功能主义的沉默权威 | AW 不强调"生存"，AW 强调"享受"和"游戏" |
| **Nike** | "每个人都有值得打的仗"——平凡人的英雄时刻 | AW 不打"仗"，AW 去"玩" |
| **Apple** | "站在科技与人文的十字路口"——复杂技术的诗意化 | AW 不讲"科技"，AW 讲"有趣的魔法" |
| **传统汽配品牌** | "让你的车更实用"——功能叠加 | AW 不讲"实用"，AW 讲"有趣" |

**AW 的叙事差异化**：不谈生存、不谈战斗、不谈科技、不谈实用。**AW 谈的是"玩"——认真玩、高级地玩、把车变成游乐场。**

## 六、叙事自检清单

| # | 自检问题 | 判断标准 |
|---|---------|---------|
| 1 | **这条内容在讲哪个章节？** | 如果答不出，说明叙事方向不清晰 |
| 2 | **用户看完之后，想做什么？** | 如果答案是"无感"，说明没有推进故事 |
| 3 | **把 AW 换成竞品，这个故事还成立吗？** | 如果成立，说明叙事不够独特 |
| 4 | **这个故事是在替用户说话，还是在替品牌说话？** | 如果是后者，重写 |
| 5 | **这个故事会让用户想"出发"吗？** | 如果不会，检查是否偏离了"鼓励出去玩"的核心引擎 |
