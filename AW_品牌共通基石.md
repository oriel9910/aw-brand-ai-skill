## 一、品牌态度

### 1.1 一句话定位

> ARCANE WARRIOR（AW）占领全球新能源车顶最后两平米，把每一场出行，变成高净值玩家的"现实游戏"。

### 1.2 终局愿景

成为户外场景的超级基础设施，成为整个时代最酷、最具势能的超级生活方式基础设施。

### 1.3 品牌精神

AW是"勇敢前行"的图腾。品牌口号：**让每一次出行都成为勇敢探索**。

### 1.4 核心态度


| 维度         | 表达                         |

## 二、产品理解

### 2.1 品类定义

AW的产品不是"载物平台"，是新能源汽车向外拓展的**"智能生态基座"**。全系无损快装，模块化拓展，让车变成一个**开放的硬件OS**。

### 2.2 技术架构

独创 **Arboris 树脉架构** + 数十项独家核心专利。无螺丝流线贴合，原车级高定姿态，拒绝任何破坏原车美感的妥协。

### 2.3 核心产品线

#### M'LinkX 车顶平台


| 维度       | 内容                                   |
| -------- | ------------------------------------ |
| **定位**   | 以M'Link平台为基石，原厂级车顶生态                 |
| **核心卖点** | 全铝轻量 / 无损快装 / 快拆拓展 / 隐藏射灯 / 适配100+车型 |
| **情感价值** | 车变成开放的硬件OS，可以外接任何你想要的户外魔法            |
| **信任支撑** | 合法合规终身免检 / 极致微缩隐形收纳 / 进得了2米地库        |


#### 无级悬停天幕


| 维度       | 内容                                       |
| -------- | ---------------------------------------- |
| **定位**   | 以悬停天幕为羽翼，女性友好、零体力消耗                      |
| **核心卖点** | 10秒展开！1秒速收 / 单手秒级悬停 / 无需工具不费体力 / 稀缺石墨烯涂层 |
| **情感价值** | 把复杂留给工业，把时间还给假期；不错过每一场山野落日               |
| **信任支撑** | 户外实测 / 日与夜晴与雨永远是您的主场 / 颜值实力双在线           |


### 2.4 目标人群

高净值人群、新能源车主、硬派越野玩家、追求身份认同与生活方式的探索者。

## 三、调性规范

### 3.1 品牌调性关键词

**高级、自信、科技感、诗意、降维打击**

### 3.2 语言风格


| 特征        | 说明                          |
| --------- | --------------------------- |
| **科技诗意**  | "以悬停天幕为羽翼""把复杂留给工业，把时间还给假期" |
| **降维打击感** | "把出行变成现实游戏""开放的硬件OS"        |
| **游戏化隐喻** | "现实游戏""高净值玩家"               |
| **身份认同**  | "鄙视链顶端""勇敢前行的图腾"            |


### 3.3 调性禁区


| ❌ 禁止             | ✅ 替代方向          |
| ---------------- | --------------- |
| 走性价比路线           | 品牌溢价、圈层认同       |
| 无聊的传统制造业语言       | 科技诗意、降维打击       |
| 堆砌参数卖点           | 用场景和情感传递价值      |
| 空洞的鸡汤/口号         | 有画面感、有态度的短句     |
| 说教式"你应该…"        | 邀请式"去…"         |
| 土味广告语（如"优雅接管山野"） | 克制、留白、有呼吸感的高级表达 |


### 3.4 高级感来源

**克制。字越少，越高级。不说"我很高级"，本身就是高级。**
