## 一、品牌态度

### 1.1 一句话定位

> ARCANE WARRIOR（AW）占领全球新能源车顶最后两平米，把每一场出行，变成高净值玩家的"现实游戏"。

### 1.2 终局愿景

成为户外场景的超级基础设施，成为整个时代最酷、最具势能的超级生活方式基础设施。

### 1.3 品牌精神

AW是"勇敢前行"的图腾。品牌口号：**让每一次出行都成为勇敢探索**。

### 1.4 核心态度


| 维度         | 表达                         |
| ---------- | -------------------------- |
| **绝不走性价比** | 享受极高品牌溢价和健康利润率             |
| **品类开创者**  | 在"新能源智能户外生态"新品类里，AW是绝对的开创者 |
| **生而全球化**  | 搭载中国新能源汽车出海超级巨浪            |
| **文化认同优先** | 前端靠极高粘性的文化认同直接转化           |
| **鄙视链顶端**  | 通过最核心KOL圈层，建立不可撼动的圈层地位     |

## 二、产品理解

### 2.1 品类定义

AW的产品不是"载物平台"，是新能源汽车向外拓展的**"智能生态基座"**。全系无损快装，模块化拓展，让车变成一个**开放的硬件OS**。

### 2.2 技术架构

独创 **Arboris 树脉架构** + 数十项独家核心专利。无螺丝流线贴合，原车级高定姿态，拒绝任何破坏原车美感的妥协。

### 2.3 核心产品线

#### M'LinkX 车顶平台


| 维度       | 内容                                   |
| -------- | ------------------------------------ |
| **定位**   | 以M'Link平台为基石，原厂级车顶生态                 |
| **核心卖点** | 全铝轻量 / 无损快装 / 快拆拓展 / 隐藏射灯 / 适配100+车型 |
| **情感价值** | 车变成开放的硬件OS，可以外接任何你想要的户外魔法            |
| **信任支撑** | 合法合规终身免检 / 极致微缩隐形收纳 / 进得了2米地库        |


#### 无级悬停天幕


| 维度       | 内容                                       |
| -------- | ---------------------------------------- |
| **定位**   | 以悬停天幕为羽翼，女性友好、零体力消耗                      |
| **核心卖点** | 10秒展开！1秒速收 / 单手秒级悬停 / 无需工具不费体力 / 稀缺石墨烯涂层 |
| **情感价值** | 把复杂留给工业，把时间还给假期；不错过每一场山野落日               |
| **信任支撑** | 户外实测 / 日与夜晴与雨永远是您的主场 / 颜值实力双在线           |


### 2.4 目标人群

高净值人群、新能源车主、硬派越野玩家、追求身份认同与生活方式的探索者。

## 三、调性规范

### 3.1 品牌调性关键词

**高级、自信、科技感、诗意、降维打击**

### 3.2 语言风格


| 特征        | 说明                          |
| --------- | --------------------------- |
| **科技诗意**  | "以悬停天幕为羽翼""把复杂留给工业，把时间还给假期" |
| **降维打击感** | "把出行变成现实游戏""开放的硬件OS"        |
| **游戏化隐喻** | "现实游戏""高净值玩家"               |
| **身份认同**  | "鄙视链顶端""勇敢前行的图腾"            |


### 3.3 调性禁区


| ❌ 禁止             | ✅ 替代方向          |
| ---------------- | --------------- |
| 走性价比路线           | 品牌溢价、圈层认同       |
| 无聊的传统制造业语言       | 科技诗意、降维打击       |
| 堆砌参数卖点           | 用场景和情感传递价值      |
| 空洞的鸡汤/口号         | 有画面感、有态度的短句     |
| 说教式"你应该…"        | 邀请式"去…"         |
| 土味广告语（如"优雅接管山野"） | 克制、留白、有呼吸感的高级表达 |


### 3.4 高级感来源

**克制。字越少，越高级。不说"我很高级"，本身就是高级。**

## 四、创始人观点与用户洞察

> 以下内容提炼自创始人招商访谈逐字稿，是品牌战略层级的核心认知；**原载《AW_品牌文案参考规则》第十章**，现迁入本文为 **第四章**，编号 **4.1～4.5** 与历史 **10.1～10.5** 一一对应。所有 AW 向产出均应与之对齐。

### 4.1 创始人的品牌信念


| 维度           | 创始人原话/观点                                         |
| ------------ | ------------------------------------------------ |
| **品牌一句话**    | "勇敢享受世界。AW所有的装备都是鼓励大家要出去玩的。"                     |

| **品牌自信**     | "很多人以为我们是国外品牌，但我们是地道的中国品牌。AW的视觉做得很洋气。"           |
| **产品的本质**    | "AW做的东西看似没有用，高端产品看似没有用，但它很有趣。一摆就是好看，你就会忍不住拿照片拍。" |


### 4.2 创始人对合作伙伴的态度


| 维度            | 创始人原话/观点                                   |
| ------------- | ------------------------------------------ |
| **AW的定位**     | "我们不是雪中送炭，我们是锦上添花。AW可以在现有用户群体上原地增值。"       |
| **门店的核心责任**   | "门店承担的责任就是要给用户一个非常好的体验感。"                  |
| **不是只会赚钱的门店** | "门店不能只关注盈利，我们更要关注用户想要什么。AW是一个非常在意用户的品牌。"   |
| **合伙人三要素**    | ①热爱生活，有户外的感觉 ②善于交朋友，有圈子 ③对某几个车型非常了解，或有相应群体 |


### 4.3 用户需求分层洞察

创始人认为，不同人群的真实需求完全不同，文案必须精准匹配：


| 人群       | 真实需求              | AW能给的           |
| -------- | ----------------- | --------------- |
| **女性**   | 防晒、拍照好看、温馨感觉      | 颜值在线、女性友好、场景氛围感 |
| **年轻人**  | 潮玩、与众不同、身份展示、自由宣言 | 代表自由、设计前沿、社交货币  |
| **家庭奶爸** | 对家庭的责任、带全家出行的安心   | 承担家庭旅行的责任、安全可靠  |
| **年长人群** | 享受人生美好时光          | 安心放心、全国门店售后保障   |


### 4.4 商业模式关键认知


| 维度       | 内容                                  |
| -------- | ----------------------------------- |
| **线上线下** | 线上只获客不成交，引导到当地门店；线下成交+服务+体验         |
| **区域保护** | 一线城市2~3家（按车型区分），小城市1家               |
| **利润模型** | 毛利率30%~~接近50%；回本周期1~~3个月；轻资产运作      |
| **增长逻辑** | 不是"复购"（同产品重复买），而是"加购"（不断买新的场景化装备）   |
| **场景覆盖** | 潮玩、骑行、过年回家、滑雪、钓鱼、亲子旅行、露营——跟生活方式紧密挂钩 |


### 4.5 对文案创作的指导原则

从创始人访谈中提炼的6条文案创作红线：


| #   | 原则                | 创始人态度                                      |
| --- | ----------------- | ------------------------------------------ |
| 1   | **说"有趣"而非"有用"**   | "看似没有用，但它很有趣"——文案要传递趣味和向往，不是功能参数           |
| 2   | **站在用户角度，不是品牌角度** | "大家都站在门店角度，很少站在用户需求角度"——文案要替用户说话           |
| 3   | **不同人群用不同话术**     | 女性/年轻人/奶爸/年长者需求完全不同——文案要精准匹配               |
| 4   | **AW代表前沿和自由**     | "代表前沿，代表设计风格风向""代表自由"——品牌调性的核心锚点           |
| 5   | **品质服务永不妥协**      | "品质、细节、服务永远都不会变"——品牌承诺可在信任型文案中反复使用         |
| 6   | **中国品牌，世界级审美**    | "很多人以为我们是国外品牌，但我们是地道的中国品牌"——品牌自信，可在品牌故事中强化 |
