# AW Product：产品与参数方向（占位骨架）

> **占位说明**：SKU 表、适配清单、技术参数真源，将在 **`AW_产品参数与SKU.md`**（待建）或你们现有 PIM/表格路径维护。当前若文档缺失，**仅可使用用户粘贴或仓库内已存在的结构化资料**，禁止编造参数。

## Agent 执行前

1. 若存在 `AW_产品参数与SKU.md`（或团队指定的参数表路径），**必须先读** 再回答。  
2. 无真源时：明确告知「当前仓库未挂载参数表，请粘贴官方规格或表格片段」。  
3. 将事实整理成 **对外口径** 时，仍须符合 **[AW_品牌文案参考规则.md](../../../AW_品牌文案参考规则.md)** 禁止虚假与夸大。

## 面向同事的一句话

帮你 **查、对、整理 AW 产品事实**（适配、尺寸、卖点是否写对了），不负责写整支广告脚本或整页电商故事线——那些分别找 **aw-video-script**、**aw-listing-creative**。

**非本子技能范围：** 海报感性大字、漫威叙事 → **aw-topic-brief**。
