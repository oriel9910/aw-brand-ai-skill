# AW Listing Creative：电商主图与详情方向（占位骨架）

> **占位说明**：主图信息层级、详情页模块顺序、卖点与信任块写法等，将在 **`AW_电商主图详情指引.md`**（待建）中维护。当前请先对齐 **[AW_品牌文案参考规则.md](../../../AW_品牌文案参考规则.md)** 的调性、禁区；产品事实以 **aw-product** 为准，避免详情里编造参数。

## Agent 执行前

1. 阅读 `AW_品牌文案参考规则.md` 中与「利益抓手、场景、身份认同」相关章节。  
2. 若已有 `AW_电商主图详情指引.md`，**优先阅读全文**。  
3. 需要小红书式标题包装时，可按第十二章 **可选** 使用 `dbs-xhs-title`（不违背 AW 禁区）。

## 面向同事的一句话

帮你理 **AW 电商页该讲什么、每块标题怎么写**——主图文案、详情模块小标题、卖点故事线；**具体像素与拍图**仍由设计同事定稿。

**非本子技能范围：** 短视频分镜、口播 → **aw-video-script**；纯品牌海报 → **aw-topic-brief**。
