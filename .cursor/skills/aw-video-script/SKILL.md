# AW Video Script：短视频脚本（占位骨架）

> **占位说明**：详细脚本结构、分镜表模板、口播字数节奏等，将在仓库根目录 **`AW_短视频脚本规范.md`**（待建）中维护；当前请先对齐 **[AW_品牌共通基石.md](../../../AW_品牌共通基石.md)** 与 **[AW_品牌文案参考规则.md](../../../AW_品牌文案参考规则.md)**（调性禁区以共通基石为准；类型 A 等见主文档）。

## Agent 执行前

1. **先读** [AW_品牌共通基石.md](../../../AW_品牌共通基石.md) — 态度、产品心智、调性禁区。  
2. **再读** `AW_品牌文案参考规则.md` 第四章起（方法论等）；禁止性价比腔与土味广告。  
3. 若仓库已存在 `AW_短视频脚本规范.md`，**优先阅读该文件全文** 再写脚本。  
4. 开头与钩子可 **可选** 协同本机 `dbs-hook`（见《AW_品牌文案参考规则》第十二章）；仍以 AW 调性优先。

## 面向同事的一句话

帮你写 **AW 调性下的短视频结构**：分镜要点、口播稿、钩子与结尾行动——需要时长、平台、产品出镜清单时告诉我。

**非本子技能范围：** 纯海报大字、电商详情模块标题 → 请用 **aw-topic-brief** 或 **aw-listing-creative**。
