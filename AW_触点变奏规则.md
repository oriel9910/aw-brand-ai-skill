# AW 触点变奏规则（Channel Adaptation Rules）

> **版本**：v1.0
> **更新日期**：2026-04-20
> **定位**：定义 AW 的**同一个品牌人格**在不同触点/渠道上的**语气变奏规则**。核心原则：人格不变，语气弹性变奏。不是"演不同的人"，而是"同一个人在不同场合的自然反应"。
> **上游依赖**：[AW_品牌人格画像.md](AW_品牌人格画像.md)（人格定义、语气光谱）、[AW_品牌叙事弧线.md](AW_品牌叙事弧线.md)（章节分配）
> **下游服务**：[AW_品牌文案参考规则.md](AW_品牌文案参考规则.md)（类型 A~F 的具体执行）

---

## 一、变奏总原则

### 1.1 核心原则：一个人，不同场合

> 想象 AW 是一个真实的人：TA 在朋友圈发的内容、在店里和客户聊天的样子、在发布会上说的话——**是同一个人，但语气自然不同**。

| 原则 | 说明 |
|------|------|
| **人格锚定不变** | 不管什么渠道，AW 始终是那个"已经到过山顶、回到城市也不解释的人" |
| **语气弹性变奏** | 在不同触点上，语气的"温度"和"距离感"可以有弹性 |
| **禁区全渠道通用** | 《品牌人格画像》中的"AW 绝不使用的语言"在所有渠道都生效 |
| **变奏有边界** | 每个渠道有明确的"允许范围"和"禁止越界" |

### 1.2 变奏维度

AW 在不同触点上的变奏，主要沿两个维度移动：

```
                    正式 ←————————————→ 随意
                      ↑                    ↑
                   电商详情页            用户社群/私域
                   品牌发布会            朋友圈互动
                   公众号长文            短视频评论区

                    冷 ←————————————→ 暖
                      ↑                    ↑
                   品牌态度声明          门店面对面
                   竞品打击话术          用户故事/证言
                   产品参数页            售后服务对话
```

## 二、各触点变奏规则

### 2.1 小红书（核心社交阵地）

| 维度 | 规则 |
|------|------|
| **人格状态** | "玩家对玩家"——AW 像一个已经玩得很溜的朋友，在分享经验 |
| **语气档位** | 2档（克制）~ 4档（锋利），以 3档（邀请）为主 |
| **叙事章节** | ①觉醒 + ④回来为主，辅以 ③在路上 |
| **人称** | 可以用"你"，但不要用"您" |
| **字数** | 标题 ≤15字，正文 200~500字 |
| **语言特征** | 可以有态度、可以挑衅、可以有"玩家黑话"（满配/裸车/姿态王） |
| **允许** | 「」强调标记、"？"制造好奇、偶尔的幽默 |
| **禁止** | 空洞口号、参数堆砌、说教式"你应该"、过度热情的推销 |
| **与类型A~F的关系** | 对应类型A（封面发布）和类型E（公众号长文）的小红书版本 |

**变奏示例**：

| 场景 | 语气 | 示例 |
|------|------|------|
| 装车效果展示 | 2档 克制 | "猛犸817拒绝「裸车」玩法" |
| 态度/反差钩子 | 4档 锋利 | "花五十万买车，周末停在商场地下二层" |
| 场景分享 | 3档 邀请 | "去出发，别去路过" |
| 用户互动/评论 | 3~5档 | "装上，走。" 或 "后视镜里，山在往后退" |

---

### 2.2 电商主图（转化核心）

| 维度 | 规则 |
|------|------|
| **人格状态** | "沉默的自信"——AW 不追着卖，用画面和短句让产品自己说话 |
| **语气档位** | 1档（沉默）~ 2档（克制），偶尔 3档（邀请） |
| **叙事章节** | ①觉醒（主图）+ ②出发（CTA） |
| **人称** | 尽量不用人称，纯画面或纯动作 |
| **字数** | 主标 ≤12字，副标 ≤18字，信任条 ≤8字 |
| **语言特征** | 极致短句、画面句、信任支撑（合法合规/适配100+车型） |
| **允许** | 数字锚定（10秒展开/100+车型）、信任条信息 |
| **禁止** | 感叹号堆砌、"爆款/秒杀"、参数罗列、品牌名重复 |
| **与类型A~F的关系** | 对应类型A（封面发布）的电商版本，遵循海报三层结构 |

**变奏示例**：

| 位置 | 语气 | 示例 |
|------|------|------|
| 主标 | 1档 沉默 | "装上，走。" / "装得下旷野，也进得去地库" |
| 副标 | 2档 克制 | "原厂级车顶生态" / "把复杂留给工业，把时间还给假期" |
| 信任条 | 2档 克制 | "全系装备 / 适配100+车型 / 全国100+授权门店" |
| CTA | 3档 邀请 | "去出发" / "即刻体验" |

---

### 2.3 电商详情页（深度说服）

| 维度 | 规则 |
|------|------|
| **人格状态** | "有底气的专家"——AW 像一个在店里一对一给你讲解的人，专业但不机械 |
| **语气档位** | 2档（克制）~ 3档（邀请），模块标题用 1~2档，场景描述用 3档 |
| **叙事章节** | ①觉醒 → ③在路上 → ⑤加购（完整叙事弧） |
| **人称** | "你"为主，但不卑不亢 |
| **字数** | 模块标题 ≤12字，模块描述 50~150字 |
| **语言特征** | 模块标题用画面句或命题句，模块描述用场景化语言，不用说明书句式 |
| **允许** | 适度参数（必须经 aw-product 核对真源）、场景化描述、用户证言 |
| **禁止** | "支持/采用/实现"工程句式密集、参数堆砌、空洞形容词 |
| **与类型A~F的关系** | 对应类型C（新品上新）的详情页版本 |

**变奏示例**：

| 模块 | 语气 | 标题示例 | 描述方向 |
|------|------|---------|---------|
| 痛点引入 | 4档 锋利 | "车顶空着，(可惜了)" | 制造不适感 |
| 产品展示 | 2档 克制 | "装得下旷野，也进得去地库" | 场景对比 |
| 技术解读 | 2档 克制 | "拒绝任何破坏原车美感的妥协" | 态度声明 |
| 场景描绘 | 3档 邀请 | "别人的车顶落灰，你的车顶落日" | 画面代入 |
| 收口 | 1~2档 | "装上，走。" | 行动号召 |

---

### 2.4 公众号（品牌深度阵地）

| 维度 | 规则 |
|------|------|
| **人格状态** | "有温度的朋友"——AW 像一个在篝火旁聊天的朋友，讲故事、聊感受 |
| **语气档位** | 3档（邀请）~ 5档（温度），以 5档为主 |
| **叙事章节** | 五个章节均可，以 ③在路上 + ④回来 + ①觉醒为主 |
| **人称** | "你"和"我们"交替使用 |
| **字数** | 800~2000字 |
| **语言特征** | 讲故事、有情感钩子、像朋友聊天，但保持克制不煽情 |
| **允许** | "嘿，AW的玩家们"开头、用户证言、品牌温度金句、创始人观点 |
| **禁止** | 鸡汤式煽情、说教、过度商业化（公众号不是卖货页） |
| **与类型A~F的关系** | 对应类型E（公众号长文） |

**变奏示例**：

| 位置 | 语气 | 示例 |
|------|------|------|
| 开头钩子 | 5档 温度 | "嘿，AW的玩家们，还记得…" |
| 产品故事 | 3档 邀请 | "把复杂留给工业，把时间还给假期" |
| 用户证言 | 5档 温度 | 真实车主故事（淘车强等） |
| 品牌温度 | 5档 温度 | "AW始终相信，好的户外装备，是有温度的" |
| 行动号召 | 3档 邀请 | "去出发" / "到店体验" |

---

### 2.5 短视频（抖音/视频号）

| 维度 | 规则 |
|------|------|
| **人格状态** | "直接的行动派"——AW 像一个在拍vlog的玩家，边做边说，不废话 |
| **语气档位** | 1档（沉默）~ 3档（邀请），以 2档（克制）为主 |
| **叙事章节** | ①觉醒（前3秒）+ ③在路上（主体）+ ②出发（结尾CTA） |
| **人称** | 第一人称"我"或省略人称 |
| **字数** | 口播 ≤60字/条，字幕 ≤15字/屏 |
| **语言特征** | 极度口语化、节奏感强、每句话都有画面，不用书面语 |
| **允许** | 口语表达、音效配合、节奏剪辑、"去…"句式 |
| **禁止** | 书面语、长句、参数念稿、AI腔（"你是否也…"） |
| **与类型A~F的关系** | 对应 aw-video-script 技能的独立规范 |

**变奏示例**：

| 时间 | 语气 | 示例 |
|------|------|------|
| 0~3秒（钩子） | 4档 锋利 | "花五十万买车，周末停在商场地下二层？" |
| 3~15秒（展示） | 2档 克制 | 画面：装车过程，无口播或极简口播 |
| 15~25秒（体验） | 3档 邀请 | "别人的车顶落灰，你的车顶落日" |
| 25~30秒（CTA） | 1档 沉默 | "装上，走。" |

---

### 2.6 线下门店（面对面体验）

| 维度 | 规则 |
|------|------|
| **人格状态** | "可靠的朋友"——AW 像一个你信任的玩车朋友，帮你参谋、帮你装 |
| **语气档位** | 3档（邀请）~ 5档（温度），以 4~5档为主 |
| **叙事章节** | ④回来（用户故事）+ ⑤加购（场景延伸） |
| **人称** | 面对面自然对话，用"你" |
| **语言特征** | 口语化、真实、本地化，可以"不高级"但要"真诚" |
| **允许** | 方言/本地化表达、真实车主故事、当场安装展示、一对一沟通 |
| **禁止** | 背话术、推销腔、过度承诺 |
| **与类型A~F的关系** | 对应类型F（门店渠道） |

**变奏示例**：

| 场景 | 语气 | 示例 |
|------|------|------|
| 迎接客户 | 5档 温度 | "来了？先看看这台（车型）装完什么样" |
| 产品介绍 | 3档 邀请 | "你平时去哪玩？我帮你配一套" |
| 安装过程 | 2档 克制 | 边装边说，不过度解释 |
| 送客 | 5档 温度 | "装好了，周末去试试" |

---

### 2.7 用户社群/私域（新增触点）

| 维度 | 规则 |
|------|------|
| **人格状态** | "兄弟/玩伴"——AW 在私域里是最放松的，可以开玩笑、可以"不高级" |
| **语气档位** | 3档（邀请）~ 5档（温度），可以偶尔超出5档进入"随意"区间 |
| **叙事章节** | ③在路上 + ④回来 为主 |
| **人称** | 自然对话，可以用"咱""大家" |
| **语言特征** | 最口语化、最放松、可以有表情包、可以"不克制" |
| **允许** | 表情包、网络用语、用户UGC转发、互动话题、非正式表达 |
| **禁止** | 发广告、群发推销、官方腔、复制粘贴品牌话术 |
| **特别说明** | 这是 AW 唯一允许"不高级"的渠道——因为私域的核心是"真实"，不是"品牌" |

**变奏示例**：

| 场景 | 语气 | 示例 |
|------|------|------|
| 日常互动 | 随意 | "周末谁去（某地）？车顶有位置的举个手" |
| 用户晒图 | 5档 温度 | "这构图…你确定不是摄影师？" |
| 新品预热 | 3档 邀请 | "下周有个好玩的东西，先给你们看看" |
| 售后跟进 | 5档 温度 | "装得怎么样？有问题随时说" |

---

### 2.8 客服/售后话术（新增触点）

| 维度 | 规则 |
|------|------|
| **人格状态** | "可靠但不卑微"——AW 帮你解决问题，但不讨好 |
| **语气档位** | 2档（克制）~ 3档（邀请） |
| **叙事章节** | 不适用（售后是解决问题，不是讲故事） |
| **人称** | "你"为主，不用"您" |
| **语言特征** | 清晰、直接、有解决方案，不说废话 |
| **允许** | 直接给方案、承诺品质服务、表达"我们帮你解决" |
| **禁止** | 官方套话（"感谢您的反馈，我们将尽快处理"）、推诿、过度道歉 |
| **特别说明** | 售后的核心是"可靠"——创始人原话"品质、细节、服务永远都不会变" |

**变奏示例**：

| 场景 | 语气 | 示例 |
|------|------|------|
| 用户反馈问题 | 2档 克制 | "收到，我帮你确认一下，稍后回复你" |
| 解决方案 | 2档 克制 | "这个问题可以到最近的门店处理，我帮你预约" |
| 服务承诺 | 3档 邀请 | "AW的品质和服务不会变，放心" |
| 回访 | 5档 温度 | "问题解决了吗？还有其他需要的随时说" |

## 三、变奏速查表

| 触点 | 默认档位 | 允许范围 | 人格状态 | 核心叙事章节 |
|------|---------|---------|---------|-----------|
| **小红书** | 3档 邀请 | 2~4档 | 玩家对玩家 | ①觉醒 + ④回来 |
| **电商主图** | 2档 克制 | 1~3档 | 沉默的自信 | ①觉醒 + ②出发 |
| **电商详情页** | 2档 克制 | 1~3档 | 有底气的专家 | ①→③→⑤ |
| **公众号** | 5档 温度 | 3~5档 | 有温度的朋友 | ③ + ④ + ① |
| **短视频** | 2档 克制 | 1~3档 | 直接的行动派 | ①→③→② |
| **线下门店** | 4~5档 | 3~5档 | 可靠的朋友 | ④回来 + ⑤加购 |
| **用户社群** | 5档 温度 | 3~随意 | 兄弟/玩伴 | ③ + ④ |
| **客服/售后** | 2档 克制 | 2~3档 | 可靠但不卑微 | 不适用 |

## 四、跨渠道一致性检查

### 4.1 一致性原则

| 原则 | 说明 |
|------|------|
| **人格不分裂** | 用户在小红书看到的 AW、在门店遇到的 AW、在详情页读到的 AW，是同一个人 |
| **语气有弹性但不越界** | 允许在不同渠道有温度差异，但不允许出现人格矛盾 |
| **禁区全渠道生效** | 《品牌人格画像》中的"绝不使用的语言"在所有渠道都生效 |
| **叙事章节互补** | 不同渠道的内容应该覆盖不同章节，形成完整叙事 |

### 4.2 一致性自检

| # | 自检问题 | 判断标准 |
|---|---------|---------|
| 1 | **如果用户今天看了小红书、明天去了门店、后天收到客服消息，TA 觉得是同一个品牌吗？** | 如果觉得"不像同一家"，说明变奏越界了 |
| 2 | **这条内容有没有触犯全渠道禁区？** | 检查是否有"品质之选/行业领先/极致/您"等违禁词 |
| 3 | **这条内容的语气档位是否在允许范围内？** | 对照变奏速查表 |
| 4 | **最近一周的内容是否覆盖了至少 2~3 个叙事章节？** | 如果全是"觉醒"或全是"在路上"，需要调整 |
